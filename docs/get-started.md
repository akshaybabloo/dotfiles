# Dotfiles

Some helpful dotfiles for your system.

## Installation

```sh
git clone https://github.com/akshaybabloo/dotfiles.git
cd dotfiles
./bootstrap.sh
```

This should list out any dependencies that are missing. You can manually install them if you want to.
